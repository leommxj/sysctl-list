About this dataset

Data comes from Linux kernel documentation files plus sysctl registrations extracted from kernel C sources across release tags.

The pipeline normalizes parameter names, scans static and dynamic registration paths, then merges documentation and source matches into a static index for the site.

Some entries may still be incomplete or wrong. Kernel docs are inconsistent in places, and some registrations or aliases are hard to resolve automatically.

Project: github.com/leommxj/sysctl-list
